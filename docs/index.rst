Build Better Software
#####################

**Under development**

*Build Better Software* provides foundational principles of software development tailored for scientists and researchers. It offers practical guidance, best practices, and illustrative examples to support effective, maintainable, and reproducible research software.

The accompanying repository, **learn-dev**, is available at:
